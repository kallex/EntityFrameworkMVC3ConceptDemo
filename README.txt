Concept demo for Entity Framework and MVC3 Application
------------------------------------------------------
- Unpolished; just to demonstrate custom "parent-child" associations and view generation

Trying out:
1. git clone git://github.com/kallex/EntityFrameworkMVC3ConceptDemo.git 
2. cd EntityFrameworkMVC3ConceptDemo.git 
3. gitupdateproject.cmd 
4. Open Visual Studio Solution 
5. Rebuild 
6. Set "WebMvc3UI" as startup project 
7. "WebMvc3UI" project's properties under "Web" tab => Start Action = 
"Project/" as "specific page" 


F5 starts and initializes empty database...
